# Contributing to Intervue Coder

Thank you for your interest in contributing to **Intervue Coder**, a project by **Suryanshu Nabheet**! This tool is designed to provide elite interview assistance, and we welcome contributions that meet our high standards of quality and performance.

## Getting Started

1.  **Fork the Repository**:
    - Fork [https://github.com/Suryanshu-Nabheet/Intervue_Coder](https://github.com/Suryanshu-Nabheet/Intervue_Coder).
    - Clone your fork locally.

2.  **Create a Branch**:
    - `git checkout -b feature/your-feature-name`

3.  **Development Standards**:
    - **Code Quality**: Maintain clean, modular TypeScript code.
    - **UI/UX**: Ensure any UI changes match the premium aesthetic (TailwindCSS).
    - **Testing**: Test all changes thoroughly, especially AI integrations and window management behavior.

4.  **Submit a Pull Request**:
    - Push your changes to your fork.
    - Submit a PR to the `main` branch of the original repository.
    - Provide a clear description of your changes.
